# Assistente Virtual Financeiro

Este é um aplicativo desktop desenvolvido em Python que funciona como um assistente virtual financeiro, ajudando usuários a gerenciar suas finanças pessoais de forma inteligente e eficiente.

## Funcionalidades Principais

- Interface gráfica moderna e intuitiva
- Análise de gastos e receitas
- Recomendações financeiras personalizadas
- Planejamento orçamentário
- Relatórios e visualizações de dados

## Requisitos

- Python 3.8 ou superior
- Dependências listadas em `requirements.txt`

## Instalação

1. Clone este repositório
2. Instale as dependências:
```bash
pip install -r requirements.txt
```

## Como Executar

```bash
python main.py
```

## Estrutura do Projeto

- `main.py`: Arquivo principal da aplicação
- `src/`: Diretório com o código fonte
- `assets/`: Recursos visuais e outros arquivos
- `data/`: Armazenamento de dados
